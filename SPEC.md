# Truchet Tile Studio — ゲームモード仕様書

## 確定仕様

| 項目 | 決定内容 |
|------|---------|
| やめるボタン | なし（デザインタブで戻る） |
| タイマー表示 | `mm:ss.ms`、スタートボタンと差し替え |
| 一時停止ボタン | シャッフルボタンと差し替え |
| オブジェクト | カーナビ三角、道幅連動、接線方向に回転 |
| 速度カーブ | `speed = base * e^(a * elapsed)`、aは小さめ |
| ハイライト | 白オーバーレイ、前方グラデーション、速度で短く薄く |
| 終了条件A | デッドエンド到達 |
| 終了条件B | タイル遷移時に遷移先が回転中 |
| ブラウザバック | 強制終了・記録なし |
| デザインタブ押下 | 一時停止メニュー（再開／最初から／やめる） |
| ループ | 一周でタイマーグレーアウト停止、速度は継続上昇 |
| ランキング | localStorage上位3位、キャンバス中央オーバーレイ、背景半透明グレー |
| スコア保存 | localStorage |

## データ設計

### 道の表現
```js
// タイル内の道はローカル座標で保持（回転に強い）
road = { edgeA: {edge, t}, edgeB: {edge, t}, warp? }
// 回転後: edgeA.edge = (edge + rot) % n
```

### グラフ構造
- ノード = (tileId, segmentIdx, side)
- タイル境界接続 = 辺番号 + 隣接タイル座標テーブル
- タイル境界でのt値反転：出口t → 入口 `1-t`

### 先読みアルゴリズム
1. 現在位置から経路を前向きに辿る
2. 既訪問ノード出現 → ループ確定
3. グリッド外到達 → デッドエンド確定
4. タイル回転のたびに再計算

## UI状態遷移

```
待機中
  └─[スタート押下]→ プレイ中
                       ├─[デッドエンド到達]→ 終了（記録あり）
                       ├─[条件B発動]→ 終了（記録あり）
                       ├─[ブラウザバック]→ 終了（記録なし）
                       ├─[デザインタブ押下]→ 一時停止
                       └─[一時停止ボタン]→ 一時停止
                                              ├─[再開]→ プレイ中
                                              ├─[最初から]→ 待機中
                                              └─[やめる]→ 待機中
```

## 未調整パラメータ（実装後に調整）
- 初速の値
- 速度上昇係数 a
- ハイライトが消える速度閾値
- ランクインアニメーション詳細

# リリース仕様（確定）

## 配布・マネタイズ

| 項目 | 決定内容 |
|------|---------|
| プラットフォーム | Android優先、Capacitor + AdMob |
| 広告種別 | リワード広告（メイン）＋インタースティシャル（3ゲームに1回） |
| コンティニュー | リワード広告視聴でコンティニュー可（ランキングは別枠扱い） |

## 画面構成

```
タイトル
  ├─ プレイ
  ├─ デイリーチャレンジ
  ├─ タイル作成（現行のデザインタブ相当）
  ├─ 図鑑
  └─ 設定
```

## タイル図鑑（図鑑）

### ページ構成
- 3ページ：四角形タイル／六角形タイル／三角形タイル

### タイル同一性（canonical ID）
- 辺上の接続点の配置 ＋ 接続ペアの組み合わせを、回転の最小表現に正規化
- 式：全回転を試して辞書順最小の文字列表現をIDとする

### 鏡像タイル
- 鏡像は**別タイル**として管理
- 図鑑UI上では鏡像同士を線でつなぎ「関連タイル」として表示
- 両方発見済みのとき「鏡像発見！」リワードを付与

### バリアント（同一タイルの派生）
- 接点シフト、ワープ接続、z-order違いは代表タイルの下にまとめて表示

### 発見・解放
- ゲームプレイ中に通過したタイルを自動登録
- 未発見タイルはシルエット表示

## スコアシステム

### 5指標
| 指標 | 内容 |
|------|------|
| 生存時間 | ゲーム終了時のタイマー値 |
| アイテム取得回数 | 速度アップ・ダウン・アクティブ合計 |
| 道カバー率 | 全コネクション中、通過済みの割合（%） |
| 最高速度 | ゲーム中の最大speed値 |
| ループ脱出回数 | ループ検出後に脱出した回数 |

### ランキング
- グリッドサイズごとに別ランキング（localStorage、上位3位）
- コンティニュー使用プレイは別枠（通常ランキングに混入しない）

## サウンド

| 項目 | 内容 |
|------|------|
| SE | タイル回転、アイテム取得、ゲームオーバー、ランクイン等 |
| BGM | ループBGM（ゲームプレイ中） |
| 設定 | SE音量／BGM音量を独立して調整（設定画面） |

## アイテム設計

### ループ中のスポーン
- ループ状態検出中は、アイテムをループ外のコネクションにスポーン
- プレイヤーのループ脱出モチベーションを高める

### アクティブアイテム（v1、初期2種）
| アイテム | 効果 |
|---------|------|
| 逆走 | 進行方向を一定時間反転 |
| タイムストップ | タイマーを一定時間停止（速度は継続） |

### アクティブアイテム取得制限
- 1・2個目：通常取得
- 3個目以降：リワード広告視聴で取得

## 未決定・要調整

- 図鑑のアンロック演出詳細
- ランクインアニメーション詳細
